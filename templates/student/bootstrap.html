Introduction

Bootstrap is a free, open-source front-end framework used for designing modern, responsive websites quickly. It includes ready-to-use HTML, CSS, and JavaScript components that make web development faster and easier.

Features of Bootstrap

  • Responsive grid system
  • Pre-designed UI components (buttons, cards, modals, navbars)
  • Cross-browser compatibility
  • Customizable themes and utilities
  • Supports Flexbox and modern CSS

Functions of Bootstrap

Bootstrap provides built-in classes and functions that help developers:

  • Create mobile-first layouts with its grid system
  • Style elements using pre-defined CSS classes
  • Add interactivity with JavaScript components (dropdowns, modals, tooltips)
  • Speed up development by reusing common UI patterns

Bootstrap Libraries

Bootstrap comes with three main parts:

  • Bootstrap CSS → Provides styling, typography, grid system, utilities
  • Bootstrap JavaScript → Provides interactivity like modals, carousels, dropdowns
  • Bootstrap Icons → A large set of scalable vector icons
